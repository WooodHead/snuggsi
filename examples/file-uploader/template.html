<file-uploader>

Custom <input type=file> element

Settings

label
Default text
title
Default dragenter & mouseover text

Slots

mimetypes
A list of acceptable mimetypes (default: A <datalist>)

Example

CSS Styling

<style>
  file-uploader {}
  file-uploader label {}
  file-uploader table {}
  file-uploader menu button {}
</style>

HTML Markup

<file-uploader

  label='Default text'
  title='Default dragenter & mouseover text'>

  <!-- optional mimetypes list -->

  <datalist slot=mimetypes id=mimetypes>
    <option capture label=PNG accept=image/png selected>
    <option capture label=PDF accept=application/pdf>
    <option capture label=JPG accept=image/jpg>
  </datalist>

</file-uploader>
