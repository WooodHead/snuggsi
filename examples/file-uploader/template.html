<file-uploader>

Custom <input type=file> element

Slots

mimetypes
A list of acceptable mimetypes (default: A <datalist>)

Example

<file-uploader>

  <datalist slot=mimetypes>
    <option accept=image/jpeg label=image-jpeg.png> jpg
    <option accept=image/png  label=image-png.png selected> png
    <option accept=application/pdf label=application-pdf.png> pdf
  </datalist>

</file-uploader>
