# Snuggsi
Sane Client Components

## Installation

Add this line to your application's Gemfile:

### Node
```bash
$ npm install snuggsi
```

### Ruby
Add this to your gemfile
```bash
gem snuggsi
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install snuggsi
```
